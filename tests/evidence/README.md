# Test Evidence（截图 + 手动 sign-off）

Visual/Feel 与 UI 类 story 的测试证据（coding-standards：这两类 gate=ADVISORY）。

- **Visual/Feel**（动画/VFX/手感）：截图 + lead sign-off。命名 `[system]-[feature]-[date].png` + 对应 `.md` sign-off。
- **UI**（菜单/HUD/屏/卡牌）：手动 walkthrough doc OR 交互测试记录。

> 渲染时刻/手感不自动化（coding-standards "What NOT to Automate"）——靠截图对比 + lead 签字。
> VFX(19)/HUD(16) 的 experiential AC（如掷骰爽感节奏）须 playtest-signoff，证据落此。
